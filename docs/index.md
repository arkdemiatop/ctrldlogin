# ctrldlogin

**Desktop application for managing multiple isolated browser profiles with unique fingerprints.**

ctrldlogin lets you create, manage, and launch multiple isolated browser profiles — each with a unique digital fingerprint. It handles proxy integration, platform spoofing, and anti-detection so every session looks like a real, distinct device.

## Who is it for?

| Use Case | Description |
|----------|-------------|
| **Privacy-conscious users** | Manage multiple online identities without cross-contamination |
| **Freelancers** | Handle multiple client accounts with separate browser environments |
| **QA testers** | Test websites in isolated browser sessions with distinct fingerprints |
| **Marketing professionals** | Manage multiple social media or ad platform accounts |

## Quick Overview

- **Isolated profiles** — each profile runs in its own browser process with a unique fingerprint
- **Deterministic seeding** — same profile name always produces the same fingerprint
- **Proxy integration** — assign proxies per-profile, WebRTC auto-matches your exit IP
- **Cross-platform** — Windows, macOS, Linux
- **Local-first** — all data stored locally, no cloud dependency
